## 🎓 使用指南

### 对于最终用户

**三步使用**:
1. 访问 https://QzuserWY.github.io/dd-mod-sorter/
2. 上传 persist.game.json 文件
3. 调整 MOD 顺序并下载

详见：`QUICK_START.md`

### 对于开发者

**本地开发**:
```bash
git clone https://github.com/QzuserWY/dd-mod-sorter.git
cd dd-mod-sorter
npm install
npm run dev
```

**构建生产版本**:
```bash
npm run build
```
