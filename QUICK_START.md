# 🚀 快速开始指南

## 在线使用（推荐）

### 访问工具
直接访问以下链接，无需任何安装：
👉 **https://QzuserWY.github.io/dd-mod-sorter/**

### 三步使用

#### 第一步：上传存档文件
1. 打开工具网页
2. 拖拽或选择您的 `persist.game.json` 存档文件
3. 工具会自动验证并读取 MOD 列表

#### 第二步：排序 MOD
选择以下任一方式调整 MOD 顺序：

**方式 A：拖拽排序**
- 直接拖拽 MOD 项目到新位置
- 最直观、最灵活的方式

**方式 B：按钮调整**
- 使用上下箭头按钮微调位置
- 适合精细调整

**方式 C：快速排序**
- 点击"升序"或"降序"按钮
- 快速按名称排列所有 MOD

**方式 D：搜索过滤**
- 输入 MOD 名称或 ID 进行搜索
- 快速定位特定 MOD

#### 第三步：下载并应用
1. 点击"下载修改后的存档文件"
2. 备份您的原始 `persist.game.json` 文件
3. 将下载的文件重命名为 `persist.game.json`
4. 替换原始文件
5. 启动游戏

## 📍 存档文件位置

### Windows
```
%APPDATA%\LocalLow\Red Hook Studios\Darkest Dungeon\profile_1\persist.game.json
```

### macOS
```
~/Library/Application Support/Darkest Dungeon/profile_1/persist.game.json
```

### Linux
```
~/.config/unity3d/Red Hook Studios/Darkest Dungeon/profile_1/persist.game.json
```

## ⚠️ 重要提示

1. **备份文件** - 修改前务必备份原始存档文件
2. **验证文件** - 确保上传的是 `persist.game.json` 文件
3. **文件名** - 下载后必须重命名为 `persist.game.json`
4. **替换位置** - 确保放在正确的存档目录

## 🆘 故障排除

### 问题：文件验证失败
**解决方案**：
- 确保选择的是 `persist.game.json` 文件
- 文件不能被损坏或加密
- 尝试使用其他存档文件

### 问题：MOD 列表为空
**解决方案**：
- 确保存档文件包含 MOD 信息
- 检查文件是否完整
- 尝试在游戏中添加一些 MOD 后重新导出

### 问题：下载后游戏无法识别
**解决方案**：
- 确保文件名为 `persist.game.json`
- 确保放在正确的存档目录
- 检查文件是否被正确覆盖
- 尝试重启游戏

### 问题：浏览器不支持
**解决方案**：
- 使用现代浏览器（Chrome、Firefox、Safari、Edge）
- 更新浏览器到最新版本
- 清除浏览器缓存后重试
